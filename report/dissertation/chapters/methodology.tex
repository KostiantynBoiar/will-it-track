We aggregate scores to a \emph{cell} --- a (species, location, time) group --- carrying support counts.

\section{Measuring transfer}
% TODO: frozen SAM 3 inference over the test split (species + generic prompts); the OFFICIAL VEval scorer.
SAM\,3~\cite{sam3} is run frozen over the test split and scored with the official evaluator (the metric
is never re-implemented), giving per-cell \texttt{pDetA}/\texttt{pAssA}/\texttt{pHOTA}~\cite{hota}.

\section{The frozen reference (leakage firewall)}
% TODO: freezing the seen species/locations + manifest; disjointness assertion.
The seen set is frozen once; every distance is computed against the training split only, so no test
information can leak into a feature.

\section{Modelling the law}
% TODO: beta / logit-link GLM, support-weighted, log(n_frames) covariate; variance partitioning; grouped CV; the reliability estimator.
Each bounded target is modelled with a support-weighted beta regression (with a \(\log(n_\text{frames})\)
covariate), variance is attributed by dominance analysis, and the predictor is validated out-of-sample
by holding out whole species and whole locations.
